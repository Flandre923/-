## 1. 介绍
这是一个关于东方夜雀食堂特殊客人开卡的菜谱查询工具，提供了开卡的简单的菜谱酒水以及价格标注，和角色的好感度解锁的物品菜谱等。

## 2. 如何使用
请使用 Edge 浏览器打开 index.html 文件即可使用。

## 3. 项目开发
此项目使用了 Vue 开发，其中 Vue.js 是 vue 的相关内容，info.js 是我页面展示要使用的数据。index.hmlt 是运行文件。
